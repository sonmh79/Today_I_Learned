## 🤔 3-step-thinking

어떤 지식을 배웠는지 더 많이 생각하기 위해 3단계 방식으로 생각하는 템플릿을 만들었습니다.

What - 그 지식이 무엇인지 한 줄 또는 두 줄 이내로 정의합니다.

How - 해당 지식은 어떻게 동작하는지 생각합니다.

Use - 지식이 어떻게 사용될 수 있을지 생각해봅니다.
